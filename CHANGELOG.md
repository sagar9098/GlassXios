## 1.0.0

* Initial release of GlassXios.
* Native iOS Liquid Glass via `UIVisualEffectView` + PlatformView.
* Adaptive `BackdropFilter` fallback for Android and Web.
* Full widget set: `GlassXMaterialApp`, `GlassXScaffold`, `GlassXAppBar`,
  `GlassXContainer`, `GlassXCard`, `GlassXBottomSheet`, `GlassXDialog`,
  `GlassXBlur`, `GlassXNavBar`.
* `GlassXConfig` with `.low()`, `.balanced()`, `.high()` presets.
* Global configuration via `GlassXProvider` InheritedWidget.
* Debug overlay for performance monitoring.
* Animated blur transitions support.
